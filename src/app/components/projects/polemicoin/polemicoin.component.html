Polemicoin

Polemicoin is a Solana-based SPL token project that combines a public voting system with token rewards. I built the full web platform and blockchain integration, enabling users to vote on proposals, earn tokens for their participation, and track the project's evolution in real time.

The solution includes automated bots for freezing accounts and executing programmed purchases according to predefined parameters, ensuring fair token distribution and transparent governance.

Note

To interact with the live site you need a Solana-compatible wallet (e.g. Solflare, Phantom, Trust).
